# SAVE RABBIT 🐇🥕

Alternatively, you can play the live version [HERE](https://instai4.github.io/SAVE-RABBIT/) 


A 3D endless runner where you play as a rabbit running on a spherical world. Jump to collect carrots and avoid hedgehogs while a hungry monster chases you from behind.



## How to Play

- **Click** or **tap** anywhere to make the rabbit jump.
- **Goal**: Run as far as possible. Your distance is shown at the top.
- **Carrots** (pink cylinders with green leaves) give you a bonus and slow down the monster slightly.
- **Hedgehogs** (black spiky creatures) will hurt you. Avoid them!
- The monster at the bottom gets faster over time. If it catches you, game over.

## Features

- Fully 3D graphics built with Three.js
- Smooth animations using GSAP (TweenMax)
- Dynamic difficulty: monster speed increases with level (every 3000 points)
- Pause, resume, and replay buttons
- Retro-style flat shading and vibrant colors
- Original soundtrack: *Antonio Vivaldi - Summer*

## Controls

| Button   | Action                          |
|----------|---------------------------------|
| Click    | Jump                            |
| Replay   | Restart the game after game over|
| Pause    | Freeze the game                 |
| Resume   | Continue playing                |

## Technologies Used

- HTML5 / CSS3
- JavaScript (ES5)
- [Three.js r80](https://threejs.org/) – 3D rendering
- [GSAP 1.19.0](https://greensock.com/gsap-1/) – animations
- Font Awesome – icons
- Google Fonts – Voltaire, Pacifico, Open Sans, Roboto
